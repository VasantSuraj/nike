Welcome to Css, there are three types of css

1.Inline Css
  whenever we want to style only one element we will go for inline css and we will write the style attribute within the opening tag !


p2.Internal Css
  whenever we want to style more than one element but less than ten we will go for internal css

3.External Css
  whenever we want to style more than 10 elements we will go for external css that external css code will be written in another css file and linked to the html page by using link tag
